Cheap Talk and Delegation

How people share information with each other?

Basic Model

A property of cheap talk is that all the messages have the same cost.

Assume that there is a receiver \((R)\) and a sender \((S)\). They have different opinion. \(S\) knows the state of the world \(t\), \(R\) does not. The state of the world is drawn from a uniform distribution on \([0,1]\), and this is common knowledge.

\(R\) has to take a decision \(y\) after listening to the message \(m\) send by \(S\).

Sender's Payoff: $$-(y-(t+b))^2$$ Receiver's Payoff: $$-(y-(t))^2$$

The problem arises as their ideal decision is different: $$y=t+b\tag{S}$$ $$y=t\tag{R}$$

How should \(S\) communicate with \(R\)? We will use the perfect Bayesian equilibrium as the solution concept.

Truth-telling

Truthful report is not an equilibrium. Suppose \(S\) plays the truthful strategy: $$m(t)=t$$ So \(R\)'s optimal policy is: $$y(m)=m=t$$ Then, \(S\) has the following expected utility: $$-\int_{0}^{1}-(t-(t+b))^2\mathrm{d}y=-b^2$$ But by sending the following report, \(S\) can receive an expected utility of \(0\): $$m(t)=t+b$$ Thus, this is not an equilibrium.

Babbling Equilibrium

This is a negative situation where communication is meaningless. Suppose that \(S\) follows the strategy: $$m(t)=c, \forall t$$ \(R\) can infer nothing from the report. So the optimal effort is given by: $$\max_{y}-\int_{0}^{1}(y-t)^2 f(t) \mathrm{d}t$$ $$\Rightarrow y^*=\mathbb{E}[t]=\frac{1}{2}, \forall t$$ \(S\) will then not have incentive to deviate either. This is the babbling equilibrium.

Meaningful Communication

Suppose \(S\) follows the following strategy: $$m(t)=\begin{cases}m_1, \text{ for }0\leq t< t_1 \\ m_2, \text{ for }t_1< t \leq 1\end{cases}$$ Then, \(R\) infers from \(m_1\) that \(0 \leq t < t_1\), and infers from \(m_2\) that \(t_1< t \leq 1\). Thus, \(R\)'s optimal action is: $$y^*(m)=\begin{cases}y_1^*=\mathbb{E}[t_1|t\in [0,t_1)]=\frac{t_1}{2}, \text{ for }m=m_1 \\ y_2^*=\mathbb{E}[t_1|t\in (t_1,1]]=\frac{t_1+1}{2}, \text{ for }m=m_2\end{cases}$$

To support this as an equilibrium, we need \(S\) sticks to his strategy. This requires that for all \(0\leq t < t_1\): $$-\left(\frac{t_1}{2}-(t+b)\right)^2\geq -\left(\frac{t_1+1}{2}-(t+b)\right)^2$$ And for all \(t_1< t \leq 1\): $$-\left(\frac{t_1+1}{2}-(t+b)\right)^2\geq -\left(\frac{t_1}{2}-(t+b)\right)^2$$

By continuity, the threshold type \(t=t_1\) must be indifferent: $$-\left(\frac{t_1}{2}-(t_1+b)\right)^2= -\left(\frac{t_1+1}{2}-(t_1+b)\right)^2$$ $$\Leftrightarrow t_1=\frac{1}{2}-b$$ This must b \(\geq 0\), so an equilibrium can only exist if: $$b\leq \frac{1}{4}$$ Hence, for a large bias, no meaningful communication can take place.

More precise information

Suppose that an three-partition equilibrium exists, then there are two thresholds \(t_1\) and \(t_2\). \(R\) will take the following actions: $$y^*(m)=\begin{cases}y_1^*=\mathbb{E}[t_1|t\in [0,t_1)]=\frac{t_1}{2}, \text{ for }m=m_1 \\ y_2^*=\mathbb{E}[t_1|t\in (t_1,t_2)]=\frac{t_1+t_2}{2}, \text{ for }m=m_2 \\ y_3^*=\mathbb{E}[t_1|t\in (t_2,1]]=\frac{t_2+1}{2}, \text{ for }m=m_3 \end{cases}$$ For any given \(t\), the induced action is the worse the farther away from t. By a similar argument as before, we have the following indifferent threshold types conditions: $$-\left(\frac{t_1}{2}-(t_1+b)\right)^2= -\left(\frac{t_1+t_2}{2}-(t_1+b)\right)^2\tag{\(t_1\)}$$ $$-\left(\frac{t_1+t_2}{2}-(t_2+b)\right)^2= -\left(\frac{t_2+1}{2}-(t_2+b)\right)^2\tag{\(t_2\)}$$ Solving for \(t_1\) and \(t_2\), we get: $$t_1=\frac{1}{3}-4b$$ $$t_2=\frac{2}{3}-4b$$ The solutions exists if: $$b< \frac{1}{12}$$

General Partition Equilibrium

The general result is that any equilibrium is partition equilibrium. This is true for any distribution of types under some regularity conditions. For a given \(b> 0\), if a partition equilibrium exists with \(K\) partition elements, then there also exists a partition equilibrium with \(K-1\) elements.

Consider a candidate equilibrium with \(K\) partition elements, then: $$y_1^*=\frac{t_1}{2}, \text{ if } m=m_1$$ $$y_k^*=\frac{t_{k-1}+t_k}{2}, \text{ if } m=m_k, \text{ for } k=2,...,K-1$$ $$y_K^*=\frac{t_{K-1}+1}{2}, \text{ if } m=m_K$$

We then have the following indifferent condition: $$-\left(\frac{t_{k-1}+t_k}{2}-(t_k+b)\right)^2= -\left(\frac{t_k+t_{k+1}}{2}-(t_k+b)\right)^2$$ $$\Leftrightarrow t_{k+1}-t_k=t_k-t_{k+1}+4b$$ So the length of the intervals increases by \(4b\). We know that the first interval has length \(t_1\), and there are \(K\) intervals, which sum to unity: $$t_1+(t_1+4b)+...+(t_1+4b(K-1))=1$$ $$\Leftrightarrow t_1=\frac{1}{K}-2b(K-1)$$ The rhs is positive only if: $$b< \frac{1}{2K(K-1)}$$ If this is the case, then an equilibrium with \(K\) partitions exists. Moreover, we can compute the largest number of partition elements as a function of \(b\): $$K^*=\frac{1}{2}\left(1+\sqrt{1+\frac{2}{b}}\right)$$

Expected Utility

Use the fact that: $$Kt_1+2bK(K-1)=1$$ and we get the length of the \(i\) th interval is: $$\frac{1}{K}-(K-1)2b+(i-1)4b$$

Let \(P^i\) denote the i^{th} interval, then \(R\)'s expected utility (under uniform distribution) can be calculated as: $$-\sum^K_{i=1}Pr(t\in P^i)(\mathbb{E}[t|t\in P^i]-t)^2$$ $$=-\sum^K_{i=1}Pr(t\in P^i)Var(t|t\in P^i)$$ By properties of uniform distribution: $$=-\frac{1}{12}\sum^K_{i=1}\left(\frac{1}{K}+2b(2i-K-1)\right)^3$$ $$=-\frac{1}{12K^2}-b^2\frac{K^2-1}{3}$$

\(S\)'s choice is biased by \(b\), so \(S\)'s expected utility: $$-\frac{1}{12K^2}-b^2\frac{K^2-1}{3}-b^2$$

General Results

For both \(S\) and \(R\), the equilibrium with highest number of action to be played is preferred before types are drawn.

Less biased senders are better:
1. The loss for a given number of induced action is smaller.
2. A larger number of actions can be induced.

Comparison to Delegation

Imagine that the receiver can alternatively delegate decision-making to the sender. A simple delegation is that the sender can take his preferred action. Then the receiver's payoff is: $$-b^2$$ If the receiver takes the action without the information, then the payoff is (variance of \(t\)): $$\frac{1}{12}$$

The receiver then prefers delegate rather than take the decision herself with the information if: $$-b^\geq -\frac{1}{12}$$ $$\Leftrightarrow b< 0.28868$$

The receiver prefers to delegate rather than communicate if: $$-b^2\geq -\frac{1}{12K^2}-b^2\frac{K^2-1}{3}$$ With \(K\) the maximally informative equilibrium, this inequality holds with: $$b< \frac{1}{4}$$ Then we can notice that whenever meaningful communication is possible, delegation is preferred to communication.
